Mouse Sensitivity Settings

Set your advanced game settings:

  1. Click the Mouse sensitivity settings preset number 1 to enable it (if it's not already enabled).

  2. Use the vertical slider to make the mouse sensitivity value (DPI resolution) the highest that you'll want to use when playing games.

  3. Define the other sensitivity values in the same way, where number 5 is the least sensitive. (If you want to disable a preset click on its number.)

  4. If your mouse can be set to change the X-axis and Y-axis separately, make sure you define your preferred sensitivity values for both axes. You can uncheck the Link X/Y resolution settings box to set the X-axis and Y-axis separately.

  5. Check the Enable notification box if you want to hear a sound when you change mouse sensitivity while gaming. You can use the Windows® Control Panel to choose the sound.
