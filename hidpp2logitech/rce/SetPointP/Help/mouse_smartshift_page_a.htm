SmartShift™ Settings

The Logitech SmartShift technology enables you to configure how the scroll wheel on your MX Revolution mouse switches between free-spin and click-to-click modes.

  • Always click-to-click engages the wheel ratchets and disables switching to free-spin mode.

  • Always free-spin disables the wheel ratchets so the wheel spins freely, and disables switching to click-to-click mode.

  • Shift modes by pressing wheel allows you to manually switch between click-to-click and free-spin modes by pressing the scroll wheel button. Selecting this option will override the setting for button 3 on the Button Settings page.

  • Auto-shift to free spin when speed is automatically engages free-spin mode when you spin the wheel faster than the speed set by the slider. Spinning the wheel slower than this speed automatically engages click-to-click mode.

You can have different settings for specific applications by checking the Enable application-specific wheel settings box and clicking Configure. For example, you could set the wheel to always be in free-spin mode when using your web browser, and set the wheel to always be in click-to-click mode when using your photo editor.
