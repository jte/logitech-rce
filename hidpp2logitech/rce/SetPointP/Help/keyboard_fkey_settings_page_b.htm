Swap F key with FN key functions

Your keyboard F keys are located on the top row of your keyboard. They have two function modes:

  • Standard F key mode. Pressing one of the F keys alone activates pre-determined functions in the application that you are currently using. The function of each F key depends on which application is active.

  • Enhanced F key mode. Pressing and holding the FN key and pressing one of the F keys carries out the enhanced feature of that key, usually printed on the F key.

However, checking the Swap F key with FN key functions box toggles the use of the FN key. So with box checked, to activate enhanced functions press the F key alone, and to access standard F key features press FN + the selected F key.
