Scroll Panel Settings

You can have different scrolling settings for specific applications. For example, you could enable inertia scrolling when using your web browser, and disable it when using your photo editor.

For each application, select it in the Select application list and choose your preferred settings for that application.

To add a new application, click the Add... button and browse for the application. You can remove an application from the list by selecting the application and clicking the Remove... button.
