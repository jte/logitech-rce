Application-Specific Button Settings

To create custom button assignments for individual applications, select an application from the list, then choose which button(s) you want to customize for that application. For each button you can select the task you would like to assign to it. The task will only be performed in the selected application.

When you are using applications not in the Select application list, the buttons on your mouse will behave according to the tasks selected in the main Button settings page.

To add a new application, click the Add... button and browse for the application. You can remove an application from the list by selecting the application and clicking the Remove... button.
