Pointer and Scrolling Settings

Optimize the tracking and scrolling behavior of your mouse for the best comfort and control. Try different settings to see what's best for you. Note that some settings may not be available for all mice.

  • Pointer speed: Use the slider to choose the speed of the pointer relative to the movement of your mouse. Set slow speed for accuracy or fast speed for increased pointer response.

  • Pointer acceleration: Choose the rate at which the pointer increases speed when moving. Set low acceleration for accuracy or high acceleration for increased pointer response.

  • Enable smart move: Check the box to make the pointer automatically jump to the active button each time you open a new window or dialog.

  • Enable pointer trails: Check the box if you require a pointer trail when you move the mouse, and drag the slider to select the length of the trail.

  • Scrolling size: Choose how many lines are scrolled with each ratchet of the scroll wheel.

  • Enable Smooth Scrolling: Check the box to make the contents of a window scroll smoother. Note that not all applications support smooth scrolling.
