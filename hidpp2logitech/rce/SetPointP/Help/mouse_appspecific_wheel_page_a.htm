Application-Specific Wheel Settings

You can have different wheel mode settings for specific applications. For example, you could set the wheel to always be in free-spin mode when using your web browser, and set the wheel to always be in click-to-click mode when using your photo editor.

For each application, select it in the Select application list and choose your preferred SmartShift™ settings for that application.

To add a new application, click the Add... button and browse for the application. You can remove an application from the list by selecting the application and clicking the Remove... button.
