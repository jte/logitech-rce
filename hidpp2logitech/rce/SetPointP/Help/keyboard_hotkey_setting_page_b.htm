Hot Key Settings

The SetPoint™ software lets you set up a hot key to replicate a combination of keystrokes or launch an application you frequently use.

Simply select the key you want to customize in the Select Hot Key section and then choose the task you want to assign to it in the Select Task section.

For example, you can select 'One-Touch Search' from the list in Select Hot Key and select a task such as 'Open Web Page' from Select Task. Once you apply these settings, pressing the One-Touch Search hot key will launch the web page that you have entered.
