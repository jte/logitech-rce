Speed And Acceleration

Choose either of these options:

  • SetPoint implementation to use the pointer speed and acceleration settings in SetPoint™ to control tracking behavior. Go to the Pointer Settings page to adjust these settings.

    If your pointing device is not supported in SetPoint, the speed and acceleration settings are usually controlled by the settings in the Windows Control Panel. The pointer speed and precision settings in the Windows Control Panel are set to neutral when SetPoint implementation is selected.

    Check the Retain OS settings box if you want to customize the settings in the Windows Control Panel for pointing devices that aren't supported in SetPoint.

-or-

  • OS implementation to use the pointer speed and precision settings in the Windows® Control Panel to control tracking behavior.
