F key Settings

The SetPoint™ software lets you set up your F keys to suit the way you work and play. By pressing the FN key and a specific F key at the same time you can carry out various tasks on your computer that would usually be several presses or clicks to get to.

Simply select the F key you want to customize in the 2. Select F key section and then choose the task you want to assign to it in the 3. Select Task section.

For example, you can select 'E-Mail' from the list in 2. Select Hot Key and select a task such as 'Open Web Page' from 3. Select Task. Once you apply these settings, pressing the E-Mail F key will launch the e-mail web site that you have entered.
