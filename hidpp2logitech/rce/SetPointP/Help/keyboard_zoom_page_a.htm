Zoom Settings

You can zoom in and out of some applications using your keyboard, to change how much you can see of documents, web pages, and so on.

Drag the Speed slider to specify how fast you zoom in and out. Set slow speed for accuracy or fast speed for quick zooming.

Drag the Acceleration slider to choose the acceleration of the zooming. Set low acceleration for accuracy or high acceleration for increased speed zooming. Using accelerated zooming means the zooming gets faster the longer you hold the button or slider.
